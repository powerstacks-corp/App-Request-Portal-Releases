# Documentation Images

This folder contains screenshots and images used in the documentation.

## Required Screenshots

The User Guide references the following images that need to be captured:

| Filename | Description | Recommended Size |
|----------|-------------|------------------|
| `user-guide-signin.png` | The sign-in page with Microsoft authentication button | 800x600 |
| `user-guide-home.png` | Home page showing hero app and featured apps | 1200x800 |
| `user-guide-browse.png` | Browse apps page with search bar and app cards | 1200x800 |
| `user-guide-request-modal.png` | The request dialog with justification field | 600x500 |
| `user-guide-my-requests.png` | My Requests page showing request history | 1200x800 |
| `user-guide-install-status.png` | Request details showing install status | 800x600 |
| `user-guide-dark-mode.png` | Settings panel with dark mode toggle | 400x300 |

## Screenshot Guidelines

1. **Browser**: Use a clean browser window without extensions visible
2. **Resolution**: Capture at 1920x1080 or higher, then scale down
3. **Content**: Use realistic but non-sensitive sample data
4. **Format**: PNG format preferred for UI screenshots
5. **Naming**: Use lowercase with hyphens (e.g., `user-guide-home.png`)

## Tools for Screenshots

- **Windows**: Snipping Tool (Win+Shift+S) or Snip & Sketch
- **macOS**: Screenshot (Cmd+Shift+4)
- **Browser**: Browser DevTools can capture full-page screenshots
- **Annotation**: Use tools like Snagit or Greenshot for adding callouts
