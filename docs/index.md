# PowerStacks App Portal for Intune

Self-service application catalog for Microsoft Intune. Enable employees to request apps, streamline approvals, and automate deployments.

## Key Features

- **Self-Service App Catalog** — Users browse and request apps from a curated catalog
- **Approval Workflows** — Multi-stage, conditional approvals with manager and group-based routing
- **WinGet Integration** — Publish apps from the WinGet package catalog directly to Intune
- **Automated Packaging** — Cloud-based Win32 packaging with PSADT v4 wrapping
- **Email & Teams Notifications** — Actionable emails and Teams bot notifications for approvers
- **Install Status Tracking** — Real-time deployment status from Intune
- **Reports & Analytics** — ROI calculator, request trends, and CSV exports
- **Auto-Updates** — Track and apply WinGet updates for published apps
