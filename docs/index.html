App Request Portal

Self-service application catalog for Microsoft Intune

About

App Request Portal enables employees to request app installations for their Intune-managed devices. Approved requests automatically add users to Azure AD groups assigned to the requested applications.

  • Browse apps from Intune (Windows, iOS, Android, macOS)
  • Request apps with business justification
  • Multi-stage approval workflows
  • Automatic Azure AD group assignment
  • Winget catalog integration with cloud packaging
  • Microsoft Teams notifications
  • Real-time Intune install status tracking
  • Portal branding and customization

  • Azure Subscription

Estimated Monthly Cost

The default deployment (B2 App Service, Basic SQL, geo-redundant storage) costs approximately $65-75/month for Azure infrastructure.

Licensing
